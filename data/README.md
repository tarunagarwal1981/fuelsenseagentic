# Data Directory

This directory contains all data files used by the FuelSense 360 platform.

## Structure

- `ports/` - Port data files (ports.json, port metadata)
- `vessels/` - Vessel data files (vessel specifications, capabilities)
- `prices/` - Price data files (fuel prices, historical pricing)
- `compliance/` - Compliance data files (regulations, ECA zones, requirements)

## Purpose

Centralized data storage allows for:
- Easy data updates without code changes
- Version control of data files
- Separation of data from logic
- Support for multiple data sources
